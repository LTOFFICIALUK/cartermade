# CarterMade

Portfolio site for Luke Carter — websites for small local businesses.

- **Domain:** https://cartermade.co.uk
- **Stack:** Next.js (App Router), TypeScript, Tailwind CSS
- **Brief:** see `BRIEF.md` and `WEBSITE-BRIEF.md`

## Develop

```bash
npm install
npm run dev
```

Open [http://localhost:3000](http://localhost:3000).

## Brand

Atelier / workshop: walnut, ivory, brass. Fraunces + DM Sans. Primary CTA: WhatsApp.
